# Website / Media Bridge

This file explains the plain meaning of "connect website paths to YouTube."

It is not a backend integration. It is not a confusing routing project. It means that after a video or resource exists, the relevant Vince Doud website page should point to it, and the YouTube description should point back to the relevant website page.

The goal is:

`watch -> learn -> start a conversation`

## Simple Rule

When a video is published:

1. Choose the 1-3 website pages where that topic naturally belongs.
2. Add a direct "Watch" or "Learn more" link, or embed the video if the page layout supports it.
3. Add the matching website page link inside the YouTube description.
4. Track the bridge in `../YouTube/PUBLISHING_TRACKER.md`.

## Do Not Bridge Yet

- Draft production packets.
- Private media reviews.
- Unlisted videos that are still under review.
- Resources with student information, private district information, or unverified claims.
